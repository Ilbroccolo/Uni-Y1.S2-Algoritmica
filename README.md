## üõ† Come compilare i sorgenti
Gli appunti sono scritti in LaTeX. Per compilarli localmente e generare il PDF:
1. Clona questo repository: `git clone https://github.com/Ilbroccolo/Algoritmica.git`
2. Apri la cartella del progetto nel terminale.
3. Compila il file principale (`main.tex`) con `pdflatex` tramite riga di comando o usando il tuo editor preferito (TeXstudio, VSCode con LaTeX Workshop, ecc.). Si consiglia di compilare due volte per allineare correttamente l'indice.
